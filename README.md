# dbmovie

这是豆瓣电影的一个 Safari 拓展，在电影详情页面的右侧显示更多信息。

![screenshots](http://i.imgur.com/HRPynzQ.png)

## Features

* IMDb 评分信息
* 电影分级信息（但是好像出了点小问题）
* Google 搜索字幕组熟肉
* kickass.so 搜索生肉
